Payment Guide

Use Aptos testnet APT transfers to pay for endpoints. Price is fixed at 0.01 APT.
Step 1 — Start the server
npm run dev

Set your .env.local values before running.

Step 2 — Trigger a 402

Call an endpoint without payment:

curl -X POST "http://localhost:3000/api/premium-data"
Step 3 — Pay with a testnet transfer

Send exactly 0.01 APT on testnet to your PAYMENT_RECEIVER address, then call the endpoint: 

export APTOS_PRIVATE_KEY=0xYOUR_PRIVATE_KEY
export PAYMENT_RECEIVER=0xYOUR_WALLET
export APTOS_AMOUNT=0.01
npm run demo:buyer
What to show in the demo

Make one request without payment (402), then another paid request. The events will appear in the dashboard at /.
